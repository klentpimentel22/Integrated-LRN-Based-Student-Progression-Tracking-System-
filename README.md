# Integrated-LRN-Based-Student-Progression-Tracking-System-

Branch Structure (to be created in your group GitHub repository):

**main** (formerly master): the most stable, deployable codebase (production-ready).


**staging:** branch off from main; contains code for the staging environment (systems & acceptance testing).


**develop:** created from main; aggregates all developers’ merged contributions for integration testing.


**feature/your-feature-name: one branch per developer or feature (e.g., feature/create-user, feature/delete-user, feature/login).
